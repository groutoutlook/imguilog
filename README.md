# imguilog
**A simple logging library with an optional ImGui console widget.**

## TODO

## Features

- Logs to the console
- Logs to the console and files at the same time
- Logs to a log file
- Colour coded logging
- Variadic functions
- Profiler functions
- Error shutdown and breaking
- ImGui console logging widget
